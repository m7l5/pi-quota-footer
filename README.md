# pi-quota-footer

Codex, Claude, GLM, and Kimi usage quotas inline in Pi's footer.

## Install

```sh
pi install git:github.com/m7l5/pi-quota-footer
```

The extension uses Pi or provider CLI credentials and never renders tokens. It shows available quota windows, reset countdowns, stale data, and warning thresholds. GLM displays only its 5-hour and weekly model quotas.

## Commands

```text
/quota          Refresh and show details
/quota-footer   Toggle the custom footer
```
